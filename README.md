## Chronono

Centralized timer on a local and offline network, with a midi trigger, to learn a bit about go

## Usage

```
Usage of ./chronono:
  -midistart string
        MIDI regex for clock start (default "FA.*")
  -midistop string
        MIDI regex for clock stop (default "FC.*")
  -p string
        http port to serve on (default "8811")
```
